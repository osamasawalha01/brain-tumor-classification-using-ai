Brain Tumor Classification using AI 🧠🔬
📌 Project Overview
This project leverages state-of-the-art Deep Learning architectures to classify brain tumors from MRI scans. Instead of a basic CNN, I implemented Transfer Learning using ResNet50 and EfficientNet to achieve superior accuracy and more robust feature extraction.

🚀 Key Features
Transfer Learning: Utilizes pre-trained weights from ResNet50 and EfficientNet for high-precision medical imaging analysis.

Multi-Model Comparison: Evaluated different architectures to find the most efficient solution for tumor detection.

Streamlit Deployment: A user-friendly web interface that allows instant MRI upload and classification.

Conda Environment: Managed with a dedicated brain_ai environment to ensure reproducibility.

🛠️ Tech Stack
Frameworks: TensorFlow, Keras.

Architectures: ResNet50, EfficientNet.

Web Interface: Streamlit.

Libraries: OpenCV, NumPy, Matplotlib.

⚙️ How to Run
Clone the Repo:

Bash
git clone https://github.com/osamasawalha01/brain-tumor-classification-using-ai.git
Activate Environment:

Bash
conda activate brain_ai
Run Application:

Bash
streamlit run app.py
